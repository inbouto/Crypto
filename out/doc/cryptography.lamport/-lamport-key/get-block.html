getBlock

fun getBlock(keyID: Int, blockIndex: Int): ByteArray

Getter for a block of the LamportKey. A block is a set of bytes, usually of the size of the output of the hash function used.

Parameters

keyID - which key to fetch the block from. Should be either 0 or 1.

blockIndex - index of the block to fetch.

Exceptions

InvalidKeyIDException - if keyID is neither 0 nor 1

See Also

LamportKey.BLOCK_SIZE

LamportKey.setBlock

Return
a ByteArray of LamportKey.BLOCK_SIZE containing the block.
