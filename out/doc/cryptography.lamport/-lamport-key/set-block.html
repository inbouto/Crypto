setBlock

fun setBlock(keyID: Int, blockIndex: Int, block: ByteArray): Unit

Setter for a block of the LamportKey. A block is a set of bytes, usually of the size of the output of the hash function used.

Parameters

keyID - which key to fetch the block from. Should be either 0 or 1.

blockIndex - index of the block to fetch.

block - block to replace the selected LamportKey block with

Exceptions

InvalidKeyIDException - if keyID is neither 0 nor 1
