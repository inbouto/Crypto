# xor

`fun ByteArray.xor(comp: ByteArray): ByteArray`

Does a bitwise XOR (exclusive OR) comparison between two [ByteArrays](#).
Both [ByteArray](#)s have to be the same size.

`comp` - [ByteArray](#) to compare with [this](#)
